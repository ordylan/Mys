# MyKPs - 我的知识点!~

这个readme不展示项目, 就聊聊我这个项目的开发历程!
可以说是我最满意的项目, 也是用的最久的项目! 我在这个项目收获最大, 我知道了长远规划的重要, 也知道重要的事只能自己来!  
正如VSCode自带ai其名"Copilot", 也只是副驾驶, 不能代替所有! 当我回想当初, 对着ai指出哪一行需要改, 结果把项目都弄坏了... 又不是一句话项目(例如"我要任务表"), 既然项目大部分规划以及设计都在自己(网上公开的用起来真的不适!), 实现就应该自己! ai只是一个辅助查文档的, 而不是代替写的!  

长远规划很重要, 例如管理学科就是后加的, 特别麻烦! 动态更新的云存档还未完成, 也是因为旧版本遗留问题太多不好合并!

最后总结出最节约时间的workflow: 项目起始全让ai写, 觉得形状可以后让ai进一步改. 当ai改坏后,不要继续让他改,后面全部自己改(哪怕你让ai写片段你自己复制进去)! 永远不要让ai改长代码! 要不然很有可能会无意碰坏已有功能!

**此项目初始化有问题, 暂时不修复(因为我已有数据,不影响我使用), 相信想用这个项目的人一定有能力解决! 记得给我PR!**  

2026年7月12日补充: 最近几天修复了许多bug, 云存档也加上了! 呃, 就这样吧!
